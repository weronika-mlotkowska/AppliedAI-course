# Problem statement

_coil-20-proc_ is a dataset that contains a gray-scale images of 20 objects. The objects were placed on motorized turntable. The turntable was rotated through 360 degrees to vary object pose with respect to fixed camera. Images of the objects were taken at pose interval of 5 degrees. This corresponds to 72 images per object. This is a 20 classes classification task.

Dataset is available here:

https://www.cs.columbia.edu/CAVE/software/softlib/coil-20.php

http://www.cs.columbia.edu/CAVE/databases/SLAM_coil-20_coil-100/coil-20/coil-20-proc.zip
